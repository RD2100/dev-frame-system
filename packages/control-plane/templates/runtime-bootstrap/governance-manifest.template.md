## Drift Detection

On every session start, the agent must:

1. Compare protected section hashes against this manifest.
2. If any hash differs, flag `governance_drift`.
3. Check for unregistered local overrides and flag `unauthorized_mutation`.
4. Escalate drift in P0 rules or Gate 0 to human review.

```yaml
drift_action:
  protected_hash_changed: escalate
  unregistered_override: reject
  local_capability_added_without_registration: warn_and_register
  non_protected_file_changed: note_only
```
